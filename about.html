Sobre mim

Tenho uma vasta experiência no desenvolvimento de projetos Back-end. Durante minha trajetória, construí diversas aplicações utilizando a stack Node.js, que inclui o framework Express.js para criação de APIs e o Knex.js como query builder para interagir com bancos de dados. Minha expertise abrange desde a criação de APIs RESTful até a implementação de autenticação e autorização, bem como a otimização de consultas e a modelagem de bancos de dados relacionais.

Tenho ampla experiência também no desenvolvimento de interfaces de Front-end. Criei várias interfaces web utilizando HTML e CSS, e também trabalhei com tecnologias modernas como React e Axios. Minhas habilidades incluem a criação de componentes reutilizáveis, a gestão de estado com React Context ou Redux, e a integração com APIs RESTful usando Axios. Estou sempre atualizada com as melhores práticas e tendências do Front-end, buscando entregar produtos de alta qualidade e uma ótima experiência para os usuários.
